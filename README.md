## ✨ Features
 
| | |
|---|---|
| 🕐 **Live time plan** | Enter a start time and a rolling-average speed; clock times and elapsed time for every point from Motala start to Motala finish recalculate as you type. |

## 🚴 How It Works
 
1. **Set your start time** — pick your wave's start time from the selector.
2. **Dial in your speed** — enter your target rolling-average speed in km/h (default: 26.5 km/h).
3. **Fine-tune each depot** — add planned stop minutes and a headwind/tailwind margin at any checkpoint.
4. **Read the schedule** — the table shows arrival clock time and elapsed time for all 10 points on the route.
The speed knob scales every leg proportionally while keeping each leg's relative pace, so the plan stays realistic even when you adjust mid-table.

## 📝 Notes
 
- **Defaults** to a 26.5 km/h rolling average; all margins and stops start at 0.
- **Times are guideline estimates** — actual conditions will vary.
- **100% client-side** — one self-contained HTML file, no backend, no tracking, no analytics. The only external request is to Google Fonts.
- **Folk-art look** with Swedish flag colours and a Dala horse 🐴 theme.
---
 
## 🛠️ Usage
 
Just open the HTML file in any modern browser — no installation, no build step, no server required.
 
```bash
open vatternrundan-planner.html
```
